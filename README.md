- Query Parameters and String Validations

## pyenv

```sh
pyenv install 3.11
pyenv global 3.11
```

## poetry

```sh
pipx install --suffix=@1.8.2 poetry==1.8.2
pipx ensurepath
source ~/.bashrc
poetry@1.8.2 --version  # Poetry (version 1.8.2)
```

## virtualenvs

```
poetry@1.8.2 env use 3.11
poetry@1.8.2 run python --version
```

## install

```sh
poetry@1.8.2 install
```
